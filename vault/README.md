# Obsidian Vault

このフォルダは Obsidian の Vault（保管庫）として使用します。

## 開き方

1. Obsidian を起動
2. 「既存のフォルダを開く」
3. リポジトリ内の `vault/` を選択

## 運用メモ

- ノートはすべて Markdown（`.md`）で保存
- 画像や添付は `vault/_assets/` 配下に保存推奨
- `.obsidian/` は個人設定のため Git では無視（リポジトリ直下の `.gitignore` 済）

詳細は `.doc/0002_Hello Obsidian/01_運用方針.md` を参照してください。
